# pokeGAN
Creats new kinds of pokemons using WGAN! (DCGAN is also supported)
## dependence
'''
cv2
tensorflow( >=1.0)
scipy
numpy
'''
## how to use
'''
cd current dir
python resize.py
python RGBA2RGB.py
python pokeGAN.py
'''
